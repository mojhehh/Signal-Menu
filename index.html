Safety & Transparency

Every file has been scanned and verified. Zero detections. Zero RATs. Zero hidden code.

We believe in full transparency. Every single file we distribute has been uploaded to VirusTotal — the industry-standard malware scanner that checks files against 60+ antivirus engines simultaneously. The results speak for themselves: 0 detections across every file, every engine, every scan.

As of February 23, 2026, not a single security vendor has flagged any Signal Safety Menu file as malicious. Below you'll find the VirusTotal scan results for every file we ship, along with explanations of common security concerns.

SignalSafetyMenu.dll

✓ 0/66

The core mod plugin. A standard .NET assembly loaded by BepInEx — it hooks into Gorilla Tag using Harmony patches. It contains no networking code, no file exfiltration, no keyloggers. It's a mod, not malware.

SignalInjector.exe

✓ 0/66

The standalone injector for users who don't use BepInEx. Injectors often get flagged by antivirus software because they write to another process's memory — the same technique used by debuggers and anti-cheat tools. Despite this, Signal's injector has zero detections.

installer.bat

✓ 0/62

A simple batch script that automates the installation — it downloads BepInEx, places the DLLs in the right folders, and configures everything. It's a plain-text .bat file you can open in Notepad and read yourself. Nothing hidden, nothing obfuscated.

SignalAutoUpdater.dll

✓ 0/66

A lightweight companion plugin that checks for new releases on GitHub and updates the menu DLL automatically. It only contacts GitHub's public API — the same one your browser uses — and writes to your plugins folder. Fully transparent, zero detections.

What is a RAT?

A Remote Access Trojan (RAT) is malware that gives an attacker remote control over your PC — they can see your screen, steal passwords, access files, or use your webcam. RATs disguise themselves as legitimate software. Signal Safety Menu is not a RAT. It has no remote access capabilities, no command-and-control server, and no data exfiltration. It’s a local game mod that only interacts with Gorilla Tag’s process.

What are False Positives?

A false positive is when antivirus software incorrectly flags a safe file as dangerous. This is extremely common with game mods because they use techniques like DLL injection and memory patching — the same methods used by actual malware. Antivirus tools can’t always tell the difference. Even popular tools like Cheat Engine trigger false positives. If 0 out of 60+ engines flag a file, you can be confident it’s clean.

What is VirusTotal?

VirusTotal is a free online service owned by Google that scans files against 60+ antivirus engines simultaneously — including CrowdStrike, BitDefender, Kaspersky, Avast, and more. It’s the gold standard for verifying file safety. You can verify the scans yourself by re-uploading any of our files at virustotal.com.

Why Open Source Matters

Signal Safety Menu is fully open source on GitHub. Every line of code is public — you can read it, audit it, compile it yourself, or have a friend check it. Open-source software is inherently more trustworthy because anyone can verify there’s nothing malicious hidden inside. We have nothing to hide, and our code proves it.
